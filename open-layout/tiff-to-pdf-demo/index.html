TIFF to PDF Converter

Load a TIFF file (local or URL), preview it, and save as PDF on A4. JPEG and CCITT G4 data is embedded unprocessed; other formats are decoded and deflate-compressed. Embedded ICC profiles are preserved. Uses wasm-vips in the browser — no server required.
