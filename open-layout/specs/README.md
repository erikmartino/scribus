# Specifications

Reference specifications for the Open Layout project.

These are living documents that describe formats, APIs, and design decisions.
Unlike `plans/`, specs are not removed when implementation is complete — they
remain as authoritative references.

## Specs

- `document-store-spec.md` — Exploded document format and HTTP store API. IDML-inspired, git/AI-friendly.
- `spread-editor-user-interface-spec.md` — DTP interaction model: object/text mode transitions, selection gestures, cursor styles, visual feedback.
