Spread Editor

A facing-pages demo built on top of the story-editor core modules.

Booting editor...

The spread prototype treats an entire story as one flowing narrative, composed continuously from the left page into the right page.

Paragraphs remain style-aware runs, so bold emphasis , italic tone , and combined styling survive edits while the text reflows.

Use this demo to experiment with page geometry first, then layer in richer features such as folios, page-level guides, and spread annotations.

This demo reuses `story-editor/lib` modules through `spread-editor/lib/story-editor-core.js` and keeps spread-specific behavior in `spread-editor/app/*`, so additional demos can share the same editing core.

Component pages: story-core and spread-layout.
