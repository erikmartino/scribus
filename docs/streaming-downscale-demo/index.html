Streaming Image Downscaler

Downscales large images on the fly using a box filter. Supports PNG (8/16-bit), TIFF (stripped/tiled, CMYK), and JPEG2000. PNG streams row-by-row; TIFF and JP2 decode fully then feed the downscaler.
